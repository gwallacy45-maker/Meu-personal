
# Meu Personal - PWA (GitHub Pages)

Arquivos prontos para enviar ao repositório `meu-personal` do usuário `gwallacy45-maker`.

## Passos rápidos para publicar no GitHub Pages

1. Crie um repositório chamado `meu-personal` no seu GitHub (ou use o site "Create a new repository").
2. No seu repositório local, envie os arquivos desta pasta para o branch `main`.
   - Você pode fazer isso pela interface web (Upload files) ou usando git:
     ```
     git init
     git add .
     git commit -m "Meu Personal PWA"
     git branch -M main
     git remote add origin https://github.com/gwallacy45-maker/meu-personal.git
     git push -u origin main
     ```
3. No GitHub, vá em *Settings* → *Pages* → *Branch* e escolha `main` como fonte (root).
4. Aguarde alguns minutos. O site estará disponível em:
   https://gwallacy45-maker.github.io/meu-personal

5. Abra esse link no Safari do iPhone e use o menu de *Compartilhar* → *Adicionar à Tela de Início*.
